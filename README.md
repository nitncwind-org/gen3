# 奈良高専 吹奏楽部 公式サイト バージョン3

## 開発詳細
### 開発環境
Node v12.x

#### フレームワーク
Vue.js
##### デザイン
Vuetify

### セットアップ
```
npm install
```

### 開発用サーバーの起動
```
npm run serve
```

### ビルド
#### 開発環境
```
npm run build
```

#### 検証環境
```
npm run pre
```

#### リリース環境
```
npm run prod
```

### Lints and fixes files
```
npm run lint
```
